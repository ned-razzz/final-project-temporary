프로젝트 관리

Vision 기반 양팔 제조 제어

  • Depth Vision 기반 재료 위치·회전 추정, 엔드 이펙터 goal pose 갱신
  • MoveIt + Cartesian path 조합하여 파지·접근·배치 경로 생성
  • Gazebo 월드 구축 및 시뮬레이션 검증 후 OpenArm 실물 전이
  • 양팔 협응 제어로 조립, 케첩 도포, 음료 전달, 픽업존 배치

SLAM 기반 협소 공간 주행 도착지 경로

  • 로봇을 실제 다각형 외형 그대로 충돌을 계산 → 좁은 통로에서 정밀 주행
  • 실시간 라이다 반영 → 동적·돌발 장애물에 즉각 회피

픽업

서빙

  • 데이터 증강으로 형광등 플리커(Flicker) 노이즈 극복
  • 다양한 위치에서 수집된 데이터셋을 기반으로 물체 위치 변화에 대응
  • Human Interaction의 결과에 따라 목표 위치(Place)에 정확히 place
  • ACT: 거대 VLA 모델 대비 데이터와 학습 시간 최적화 & Jittering 없는 제어

감정 분석 기반 모객

  • MediaPipe FaceLandmarker로 얼굴 특징을 추출하여 2차원 좌표로 감정 점수화
  • Russell V-A 모델 기반 감정을 분석해 게임 NPC식 퍼널로 단계적으로 친밀도 형성

객체 추적 기반 고객 추종

  • DBSCAN으로 군집화해 고객 무리를 탐지
  • ArcFace 얼굴 임베딩 재식별로 추적이 끊겨도 동일 고객을 복구
  • 추적 좌표를 거리 비례 PD 제어로 변환해 ROS2 기반 실로봇 추종 주행
